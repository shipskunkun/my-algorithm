### 1. 最长不含重复字符的子字符串

请从字符串中找出一个最长的不包含重复字符的子字符串，计算该最长子字符串的长度。

 

示例 1:

输入: "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。
示例 2:

输入: "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。
示例 3:

输入: "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。
 
 
 
### 第一遍写法：

	思路：遍历s
	起始条件，low = 0, high = 1
	word = (low, high)
	如果 high 字符存在，low 左移到 map[high]
	使用map存储字母的下一个位置
	
 
 可以优化的地方:   
 
 查找 字符是否存在，可以使用 map.has  
 
 不需要保存 word，我们只需要 high 和 low 即可  
 
 初始化，不一定high 需要从 1 开始

```javascript
var lengthOfLongestSubstring = function(s) {
    if(s.length <= 1) {
        return s.length
    }
    let low = 0;
    let high = 1;
    let word = [s.charAt(0)];
    let maxLength = word.length;
    let map = new Map();
    map.set(s.charAt(0), 1);
    while(high < s.length) {
        let cur = s.charAt(high);
        let index = word.indexOf(cur);
        if(index > -1) {
            low = map.get(cur);
        }
        map.set(cur, ++high);
        word = s.substring(low, high);
        maxLength = Math.max(maxLength, word.length);
    }
    return maxLength
};
```

### 改良后的map:



var lengthOfLongestSubstring = function(s) {
    if(s.length <= 1) {
        return s.length
    }
    let low = 0;
    let high = 0;
    let maxLength = 0;
    let map = new Map();
    while(high < s.length) {
        let cur = s[high];
        if(map.has(cur)){
            low = map.get(cur)
        }
        map.set(cur, ++high);
        maxLength = Math.max(maxLength, high - low);
    }
    return maxLength
};



### 2. 有效的字母异位词


给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词。

示例 1:

	输入: s = "anagram", t = "nagaram"
	输出: true
示例 2:

	输入: s = "rat", t = "car"
	输出: false
	
	

### 解题思路

方法一：先将两个字符串转成数组，然后排序，比较是否相同

时间复杂度：O(nlogn)

方法二：统计字符串中每个单词出现的次数，看两个字符串，每个字母出现的次数是否相同

时间复杂度：O(n)

空间复杂度 O(1) O(26)==O(1) 表大小不变复杂性不变


### 方法一

```javascript
var isAnagram = function(s, t) {
    let s_new = Array.from(s).sort().toString();
    let t_new = Array.from(t).sort().toString();
    return s_new == t_new;
};

```

var isAnagram = function(s, t) {

### 方法二

```javascript
/**
 * @param {string} s
 * @param {string} t
 * @return {boolean}
 */
var isAnagram = function(s, t) {
    let map = new Map();
    for(let i of s) {
        if(!map.has(i)) {
            map.set(i, 1)
        }
        else {
            map.set(i, map.get(i) + 1)
        }
    }
    for(let j of t) {
        let num = map.get(j);
        if(!num) {
            return false
        }
        else if(num > 1) {
            map.set(j, num - 1)
        }
        else if(num == 1) {
            map.delete(j)
        }
    }
    return map.size == 0
};
```

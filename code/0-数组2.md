### 217. 存在重复元素
给定一个整数数组，判断是否存在重复元素。

如果任意一值在数组中出现至少两次，函数返回 true 。如果数组中每个元素都不相同，则返回 false 。


### 解题思路

思路：
1. set 长度是否等于 arr
2. 键值对，hash表
3. 排序，左右是否相等


### 代码一

```javascript
var containsDuplicate = function(nums) {
    let set = new Set(nums);
    return set.size != nums.length;
};
```

### 代码二

```javascript
/**
 * @param {number[]} nums
 * @return {boolean}
 */
var containsDuplicate = function(nums) {
    let obj = {};
    for(let i = 0; i < nums.length; i++) {
        if(!obj[nums[i]]) {
            obj[nums[i]] = true
        }
        else {
            return true
        }
    }
    return false
};
```

```javascript
var containsDuplicate = function(nums) {
    let arr = nums.sort((a, b) => a-b);
    for(let i = 0; i < arr.length - 1; i++) {
        if (arr[i] == arr[i+1]) {
            return true
        }
    }
    return false
};
```



### 384. 打乱数组
打乱一个没有重复元素的数组。

 

示例:

	// 以数字集合 1, 2 和 3 初始化数组。
	int[] nums = {1,2,3};
	Solution solution = new Solution(nums);
	
	// 打乱数组 [1,2,3] 并返回结果。任何 [1,2,3]的排列返回的概率应该相同。
	solution.shuffle();
	
	// 重设数组到它的初始状态[1,2,3]。
	solution.reset();
	
	// 随机返回数组[1,2,3]打乱后的结果。
	solution.shuffle();





### 解题思路
洗牌算法能保证，对于生成的排列，每一个元素都能等概率的出现在每一个位置。

数组长度为n， 先从n个数据中，随机选取一个元素，与最后一个元素交换

每个元素被选中的概率是 1/n

从剩下长度的 n-1 元素中随便取一个，与倒数第二个元素交换，第一次没有被选中的概率为 n-1/n 

第二次被选中的概率为 1/n-1 , 所以概率仍然是 (n-1)/n * 1/(n-1) = 1/n

所以每一个元素出现在每一个位置的概率，都是 1/n 

### 方法一，每次确定最后一个元素

```javascript
/**
 * @param {number[]} nums
 */
var Solution = function(nums) {
    this.arr = nums;
};

/**
 * Resets the array to its original configuration and return it.
 * @return {number[]}
 */
Solution.prototype.reset = function() {
    return this.arr;
};

/**
 * Returns a random shuffling of the array.
 * @return {number[]}
 */
Solution.prototype.shuffle = function() {
    let nums = [...this.arr];
    let n = nums.length - 1;
    while(n >= 0) {
        let index = parseInt(Math.random() * (n+1) );
        [nums[index], nums[n]] = [nums[n], nums[index]];
        n--;
    }
    return nums;
};

/**
 * Your Solution object will be instantiated and called as such:
 * var obj = new Solution(nums)
 * var param_1 = obj.reset()
 * var param_2 = obj.shuffle()
 */
```

### 3. 三数之和


给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有满足条件且不重复的三元组。

注意：答案中不可以包含重复的三元组。

 

示例：

	给定数组 nums = [-1, 0, 1, 2, -1, -4]，
	
	满足要求的三元组集合为：
	[
	  [-1, 0, 1],
	  [-1, -1, 2]
	]

### 解题思路

先给数组排序，然后确定第一个数，剩下两个数和值如果是第一个数的负值，满足条件

确定第一个数，而且是有序数组后，三数之和问题转为两数之和

记录几个错误版本:
1. 出现重复、但是几个数字顺序不同的结果
2. 第二个数重复
3. 当数组中有负数时候的排序
4. 超出时间限制

### 代码

```javascript
/**
 * @param {number[]} nums
 * @return {number[][]}
 */
var threeSum = function(nums) {
    nums.sort((a, b) => a - b);
    console.log(nums);
    let result = [];
    for(let i = 0; i < nums.length - 2; i++) {
        if(i > 0 && nums[i] == nums[i-1]) continue;  // 说明第一个数重复
        let j = i + 1;
        let k = nums.length - 1;
        while(j < k) {
            let sum = nums[i] + nums[j] + nums[k];
            if(sum == 0 ) {
                result.push([nums[i], nums[j], nums[k]]);
                while(nums[j] == nums[j+1]) {j++} // 说明第二个数重复
                while(nums[k] == nums[k-1]) {k--} // 说明第三个数重复
                j++;
                k--;
            }
            if(sum < 0 ) {
                j++
            }
            if(sum > 0) {
                k--
            }
        }
    }
    return result
};
```

### 5. [56. 合并区间](https://leetcode-cn.com/problems/merge-intervals/)

给出一个区间的集合，请合并所有重叠的区间。

 

示例 1:

```
输入: intervals = [[1,3],[2,6],[8,10],[15,18]]
输出: [[1,6],[8,10],[15,18]]
解释: 区间 [1,3] 和 [2,6] 重叠, 将它们合并为 [1,6].
```


示例 2:

```
输入: intervals = [[1,4],[4,5]]
输出: [[1,5]]
解释: 区间 [1,4] 和 [4,5] 可被视为重叠区间。
```


注意：输入类型已于2019年4月15日更改。 请重置默认代码定义以获取新方法签名。

 

### 思路：

遍历数组，相邻的两个数组，如果 arr[0].max <= arr[1].min  合并

否则，i++

错误在哪？首先，需要对数组，按照第一项升序排列

第二，合并  [1, 4] [2, 3] ， 需要删除 [2, 3]，直接在原数组上操作代价比较大，不能在原数组上操作，需要重新 把合并元素放在新数组中

### 代码

```js
/**
 * @param {number[][]} intervals
 * @return {number[][]}
 */
var merge = function(intervals) {
    if(intervals.length <= 1) return intervals;
    //第一项升序排列
    intervals = intervals.sort(function(a, b) {
        return a[0] - b[0]
    })
    //先把第一项放到结果数组中
    let result = [intervals[0]];
    for(let i = 1; i < intervals.length; i++) {
        //每次与已合并的最后一项比, 决定是合并还是，新增
        if( intervals[i][0] <= result[result.length - 1][1]  &&   intervals[i][1] > result[result.length - 1][1]) {
            result[result.length - 1] = [result[result.length - 1][0], intervals[i][1]]
        }
        else if(intervals[i][0] > result[result.length - 1][1]) {
            result.push(intervals[i])
        }
    }
    return result;
};
```
